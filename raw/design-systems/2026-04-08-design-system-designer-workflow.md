# Design System Designer's Claude Code Workflow

Author: Dhika Endi Astowo. Publication: Design Systems Collective.

CLAUDE.md as "AI brain" — contains: tech stack, folder org, design tokens (4px base: 4/8/16/24/32/48), color tokens, typography (Inter), component standards (WCAG 2.1 AA, hover/focus/active, dark mode). Natural language component generation: "Create Button with primary/secondary/ghost variants, sm/md/lg sizes, loading state." Conversational iteration replaces code editing. Figma MCP for design-to-code. CLAUDE.md acts as automated quality gate. Stack: React + TypeScript + Tailwind + Atomic Design + Storybook. Not enterprise-ready without InfoSec approval. Best for: prototyping, small teams, internal tools.
