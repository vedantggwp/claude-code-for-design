# Lenny's Podcast — Figma Team Demos Roundtrip Workflow

Host: Claire Vo. Guests: Gui Seiz (Figma Design), Alex Kern (Figma Engineering).

Key findings: Linear workflows are dead — bidirectional loop is the future. Production code pulled INTO Figma as editable components. Multi-state export (loading, error, success screens at once). AI shifted effort upstream (planning/strategy) and downstream (craft/QA), eliminated rushed middle execution. Custom Claude Code skills for pre-flight checks, linting, CI/CD monitoring (Buildkite). Codebase structure is prerequisite — messy repos degrade AI output. Design drift becomes nearly impossible when files stay synced. Engineers shift from execution to orchestration. AI explores unfamiliar codebases by querying structure automatically.
