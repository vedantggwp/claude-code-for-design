# Felix Lee — Master Claude Code + Figma MCP (50min Video)

Interviewed by Peter Yang. Trained thousands of designers. Demos: 3D animated world map in one shot, landing page in 15min, FigJam flowchart to game prototype. Setup: claude mcp add --transport http figma https://mcp.figma.com/mcp. Workflow: copy Figma frame link → paste into Claude with "Build this design" → iterate → export back to Figma. FigJam-to-game shows rough wireframes work as input (not just pixel-perfect designs). Code-to-Figma export at 40:19 enables visual feedback loops.
